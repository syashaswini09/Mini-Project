## Project Description

The Smart Parking Allocation System uses a Greedy Algorithm to allocate the nearest available parking slot to incoming vehicles, reducing waiting time and improving parking efficiency.

## Technologies Used

- Python
- Greedy Algorithm
- GitHub

## Output

The system allocates parking slots as:

V1 → S1  
V2 → S2  
V3 → S3  
V4 → S4  
V5 → S5  
V6 → S6  
V7 → S7
