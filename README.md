# Smart Parking Allocation System Using Greedy Algorithm

## Objective
Allocate parking slots efficiently using a Greedy Algorithm.

## Features
- Fast parking allocation
- Reduced waiting time
- Efficient slot utilization

## Technologies Used
- Python
- VS Code
- GitHub

## Algorithm
Greedy Algorithm selects the nearest available parking slot for each incoming vehicle.
